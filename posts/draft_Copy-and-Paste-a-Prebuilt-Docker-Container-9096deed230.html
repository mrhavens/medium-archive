Copy and Paste a Prebuilt Docker Container


As part of a fun exercise in a previous story about reverse engineering Docker images, I generated some unicode for my readers to copy and paste into the command line.

When I first wrote this, I thought there might be all sorts of fun to be had, because it opens up several interesting doors for replicating a container.

For example, QR codes can contain up to 7089 characters.

The image has been made functional by adding a small binary. The assembly source code is available here.

Since this image is so small, we won’t need to build or pull it. We can just show off our command-line skills by copying and pasting the entire container right into our Docker environment with the snippet below.
