Programming Languages Glossary

A comprehensive guide to understanding the features and uses of popular historical and contemporary programming languages

Are you curious about the origins and uses of popular programming languages? Do you want to gain a deeper understanding of the features and capabilities of each language?

Programming languages have come a long way since their inception, some of them are widely used today while others are mostly used in specific domains or were replaced by newer languages. In this glossary, you will find a comprehensive guide to understanding the features and uses of popular programming languages, both historical and contemporary. From Ada to Python, you will learn about the origins, capabilities, and common use cases of each language. Whether you’re a beginner or an experienced developer, this glossary is an essential tool for expanding your knowledge of programming languages.

  1. A-2: A historical high-level programming language that was developed in the 1950s. It was designed for business applications and it emphasizes the structured programming paradigm. It had features that allowed it to handle large amounts of data, making it suitable for applications that required processing big sets of information. Though A-2 isn’t as prevalent today as other languages from that time period, it still has some community and resources for those who are interested in it.
  2. APL (A Programming Language): A high-level programming language designed for mathematical and numerical computations. It uses a unique symbol-based syntax, which can take some time to learn but can allow for concise and powerful code once mastered. It has a wide array of built-in functions and operators for working with arrays and matrices and is often used in scientific and financial applications.
  3. Ada: A high-level programming language developed in the 1970s by the US Department of Defense. It was designed to be a safe and reliable language for use in critical systems such as avionics and air traffic control systems. Ada is known for its strong typing, formal verification, and automatic memory management features, which help prevent common programming errors.
  4. Algol (ALGOrithmic Language): A family of imperative programming languages first developed in the 1950s. It is considered to be one of the first high-level programming languages, and its syntax and structure have been influential in the development of many other programming languages. Algol is known for its structured programming constructs, such as the use of block structures and the “begin-end” construct, which help make code more readable and maintainable.
  5. Assembly: A low-level programming language that is used to directly control a computer’s hardware. Assembly code is a human-readable representation of the machine code that a computer’s processor can execute. Assembly language programming can be difficult and time-consuming, but it can also provide a level of control and optimization that is not possible with higher-level languages.
  6. B: A systems programming language developed at Bell Labs in the 1970s. It was designed as a simplified version of the BCPL language and was influential in the development of the C programming language. B is known for its small number of keywords and simple syntax, which makes it easy to learn and use for systems programming tasks.
  7. BASIC (Beginner’s All-Purpose Symbolic Instruction Code): is a high-level programming language designed to be easy to learn and use. It was developed in the 1960s and was one of the first programming languages to be widely available for home computers. BASIC is known for its simple and clear syntax, which makes it well-suited for educational and introductory programming.
  8. C: A general-purpose programming language developed at Bell Labs in the early 1970s. It is known for its efficiency, which makes it well-suited for systems programming tasks such as operating system development, device drivers, and embedded systems. C is also a popular language for developing applications that need to be highly optimized, such as video games and simulations.
  9. C#: A modern, object-oriented programming language developed by Microsoft as part of the .NET framework. It is similar to Java in many ways, but it is designed to be more tightly integrated with the Windows operating system. C# is often used for developing Windows desktop and web applications, as well as for building video games using the Unity engine.
  10. C++: is an extension of the C programming language that was developed in the early 1980s. It is designed to be an efficient and powerful language for systems programming tasks and for developing applications that need to be highly optimized. C++ is also an object-oriented programming (OOP) language, and it is often used in applications that require a high degree of control over the hardware and low-level system resources.
  11. COBOL (Common Business-Oriented Language): is a high-level programming language designed specifically for business applications, such as financial and administrative systems. It was one of the first programming languages to be widely used in industry, and it is still in use today in many legacy systems. COBOL is known for its verbose and structured syntax, which makes it well-suited for writing large and complex programs, such as accounting systems and payroll software.
  12. CPL (Combined Programming Language): A programming language developed at the University of Cambridge in the early 1960s. It was an early attempt to create a unified programming language that could be used for both scientific and non-scientific applications. CPL was an ancestor of the B and C programming languages, and its main feature is that it combined features from both ALGOL and Fortran.
  13. FORTRAN (FORmula TRANslation): A general-purpose, imperative programming language that is especially suited to numeric computation and scientific computing. It was developed by IBM in the 1950s and was one of the first high-level programming languages. FORTRAN is known for its powerful array-handling and matrix manipulation capabilities, which made it the language of choice for many scientists and engineers for several decades.
  14. GEDDES: A programming language developed by a company with the same name, primarily for the natural resource industry. It is a domain-specific language that provides commands for data manipulation and graphing that is useful in geological and environmental analysis.
  15. Go (Golang): A programming language developed at Google in the early 2000s. It is a compiled, statically-typed language designed for high-performance and large-scale systems, and it is known for its simplicity and clear syntax. Go is also designed with concurrency in mind and it provides built-in support for multi-threading. It is popular for building scalable network services and backend systems.
  16. JOVIAL (Jules’ Own Version of the International Algebraic Language): A high-level programming language developed in the 1950s and 1960s for use in the aerospace and defense industries. It was designed to be a more efficient and powerful version of Algol and it was used primarily in the development of avionics systems.
  17. Java: A general-purpose, object-oriented programming language developed by Sun Microsystems in the 1990s. It is known for its “write once, run anywhere” philosophy, which allows Java programs to run on a wide variety of platforms without modification. Java is widely used for developing enterprise applications and for building Android mobile apps.
  18. JavaScript: A high-level, interpreted programming language that is widely used for creating interactive and dynamic web pages. It is also used in web and mobile app development, desktop scripting, and even some form of backend development. JavaScript is a popular language and runs in every browser, without needing any additional software.
  19. Kotlin: A cross-platform, statically-typed programming language that is similar to Java in many ways. It was developed by JetBrains in 2011 and was officially adopted as a first-class language for Android app development by Google in 2017. Kotlin is designed to be more concise and expressive than Java, and it includes features such as null safety, extension functions, and coroutines.
  20. Lisp (LISt Processing): A family of programming languages that is known for its simple, yet powerful, list-based syntax. It was one of the first high-level programming languages and it has been influential in the development of many other programming languages, such as Scheme, Common Lisp, and Clojure. Lisp is known for its powerful and flexible programming paradigm known as symbolic computing, which allows the representation of code as data and operations on this data in a program. Lisp is widely used for research and artificial intelligence projects.
  21. MATH-MATIC: A computer programming language and system developed by the U.S. Air Force in the 1960s. It was intended to be used for mathematical modeling, simulations, and data analysis. The system included a library of mathematical and statistical functions, as well as a data-handling system, making it a powerful tool for mathematical research.
  22. Modula-2: A procedural programming language developed by Niklaus Wirth in the 1970s. It was an extension of the Pascal language, adding features such as separate compilation, stronger type-checking, and improved support for low-level programming. Modula-2 was intended to be a more efficient and safer alternative to Pascal for systems programming tasks and was used in the development of operating systems, compilers, and other system software.
  23. Objective-C: An object-oriented programming language that is widely used for developing applications for Apple’s macOS and iOS operating systems. It is a high-level language that is designed to be easy to use and read, and it is known for its dynamic binding, which allows for the late binding of objects and methods. Objective-C is a Smalltalk-style object-oriented language with a Smalltalk-inspired syntax.
  24. PHP (Hypertext Preprocessor): A widely-used, open-source scripting language that is especially suited for web development. It is a server-side language, which means that the code is executed on the server before the results are sent to the client’s web browser. PHP is often used to create dynamic web pages and is integrated with popular web technologies such as HTML, CSS, and JavaScript.
  25. PL/I (Programming Language One): A high-level, general-purpose programming language that was developed by IBM in the 1960s. It was intended to be a single language that could be used for both scientific and business applications. PL/I is known for its rich set of data types and powerful built-in functions, which makes it well-suited for large and complex programming tasks.
  26. Pascal: A high-level, imperative programming language that was developed by Niklaus Wirth in the 1970s. It was designed to be simple, efficient, and easy to learn, and it was widely used in education and for systems programming tasks. Pascal is known for its strong type-checking, which helps to prevent common programming errors, and its structured programming constructs, which make code more readable and maintainable.
  27. Prolog (Programming in Logic): is a logic programming language that is widely used for artificial intelligence and cognitive computing applications. It is based on the principles of formal logic, and it uses a unique syntax that allows for a natural representation of relationships and facts. Prolog’s main strength is its ability to solve problems that involve searching and pattern matching over large sets of data.
  28. Python: A high-level, interpreted programming language that is designed to be easy to read and write. It is known for its simple and clear syntax, which makes it a popular choice for beginners and for scripting tasks. Python is widely used for web development, scientific computing, data analysis, artificial intelligence, and many other fields. It has a large and active user community and a wealth of libraries and frameworks for various tasks making it a versatile language. Python’s simplicity allows for fast development and ease of maintenance, making it a popular choice for many companies.
  29. R: A programming language and software environment for statistical computing and graphics. It is widely used for data analysis, data visualization, and statistical modeling, and it has a strong community of users in academia, industry, and government. R has a large set of packages that covers areas of data manipulation, statistical modeling, and machine learning, thus making it an attractive tool for data scientists and statisticians.
  30. Ruby: A high-level, interpreted programming language that is designed to be easy to read and write, it was designed to be similar to natural language. It is an object-oriented language and it is known for its use of blocks, which are chunks of code that can be passed as arguments to methods. It is known for its rails framework, which simplifies web development and provides a structure for web application development.
  31. SIMULA: One of the first object-oriented programming languages, which was also designed for simulation tasks. SIMULA was developed in the 1960s and it was used in many different fields, including system dynamics modeling, telecommunications, and computer science education. The SIMULA67 version of the language was particularly influential, as it introduced many of the concepts that later became associated with object-oriented programmings, such as classes and objects.
  32. SNOBOL (StriNg Oriented and symBOlic Language): A string-oriented programming language that was designed for text processing and symbolic manipulation. It was developed in the 1960s and it was widely used for text processing, natural language processing, and data validation. SNOBOL has unique pattern matching and string manipulation capabilities that make it well-suited for text processing tasks.
  33. Smalltalk: A pure object-oriented programming language that was created at Xerox PARC in the 1970s. It is an interactive, dynamically-typed language and the first to feature a complete class library. Smalltalk is known for its elegant, readable, and simple syntax and for its innovative, fully integrated development environment, which allows developers to explore and debug their code in real-time.
  34. Swift: A modern, object-oriented programming language developed by Apple as a replacement for Objective-C. It is designed to be easy to learn and use, and it includes features such as closures, generics, and type inference. Swift is known for its strong type-checking and its automatic memory management, which makes it well-suited for developing iOS, macOS, and watchOS applications.
  35. Visual Basic (VB): An event-driven, object-oriented programming language that was developed by Microsoft in the 1990s. It is widely used for developing desktop and web applications, as well as for automating tasks in the Windows operating system. Visual Basic is known for its simple and clear syntax, as well as for its powerful built-in libraries and drag-and-drop development environment.
