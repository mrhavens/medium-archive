The Essential Python Newsletters for 2023

The inside track on all things Python — from AI breakthroughs to Raspberry Pi projects and more — delivered directly to your inbox through curated newsletters.
Once you stop learning, you start dying.” — Albert Einstein

As the co-founder of a Makerspace community for engineers and designers, I’ve seen firsthand the popularity and power of Python for rapid prototyping and AI development. During my past research work in affective computing, Python was my go-to for leveraging cutting-edge AI libraries and tools.

Although I’m not actively conducting academic research now, I still dabble in passion projects involving AI and machine learning. Python remains an ideal choice for quickly testing ideas and standing up proof of concepts. The language continues leading the way in emerging AI applications.

And through my connections in the Makerspace movement, it’s clear Python adoption is accelerating across industries. Our members use it for everything from Internet of Things devices to deep learning models.

While I don’t get to write Python daily anymore, I still value keeping up with the latest advancements and best practices. The pace of change in AI and the widespread embedding of Python gives me plenty to learn.

Curated Python newsletters help me stay in the loop as efficiently as possible. With domain experts sorting the signal from the noise, I can keep my skills sharp and discover new potential applications — despite my busy schedule.

Here are some of my favorite Python newsletters for developers at any level of experience. Whether you’re exploring AI for the first time or are a grizzled Python veteran like me, you’re sure to find these resources valuable.

Awesome Python Weekly

If you want a quick weekly rundown of all things Python, Awesome Python Weekly fits the bill. Every Friday, you’ll get a concise roundup of news briefs, new libraries, upcoming events, and more. Consider it your Friday morning shot of Python to start the weekend off right.

Better Developers

Reuven M. Lerner’s weekly Better Developers newsletter lives up to its name. With a single article sent out every Monday, Reuven dives deep into a Python topic, explaining complex concepts in an accessible way. He also provides actionable tips to improve your coding. If you retain information better with focused, serialized content, this one’s for you.

Bite Python

Curated by long-time Pythonista Muhammad Yasoob, Bite Python delivers easily digestible Python news and tutorials to your inbox weekly. The informative, entertaining content provides newbies and experts alike with an essential Python knowledge boost.

Coffee Bytes

Need help jumpstarting your workday? Pour yourself a cup and subscribe to Coffee Bytes. This Medium newsletter by Better Programming concisely summarizes need-to-know programming content, with a special focus on Python. You’ll breeze through entertaining and enlightening pieces chosen to make you more productive.

Powerful Python

Want to expand your Python skills? With new learning materials delivered daily, Powerful Python will supercharge your knowledge. Follow along with hands-on tutorials and in-depth explainers covering everything from machine learning to web frameworks.

PSF Newsletter

Stay in the loop with the latest happenings in the Python community via the Python Software Foundation’s official newsletter. Issued twice monthly, the newsletter keeps you up-to-date on Python news, releases, events, and more straight from the source.

Pycoder’s Weekly

With over 100,000 subscribers, Pycoder’s Weekly is one of the most popular Python newsletters around. The weekly publication summarizes top Python news, articles, tools, and more — with TL;DR summaries for blazing through it.

PYnative Newsletter

Founded by Python guru Vishal Hule, the PYnative newsletter has it all: curated tutorials, exercises, community forums, and weekly Python news/tips. Two-way communication and retention-focused training make this one of the best for learning.

Python Morsels

Bite-sized brilliance. Python Morsels sends weekly Python tips, tricks, and exercises from expert trainer Dan Bader straight to your inbox. The focused lessons teach essential Python skills through hands-on problem solving practice.

Real Python

With several issues per week, Real Python’s newsletter is a Python powerhouse. Get tutorials, guides, and cutting-edge insights to take your Python to the next level. Content is beginner-friendly but also great for seasoned developers.

STX Next

STX Next’s bi-monthly newsletter by Lukas Gmys showcases the best and most relevant Python content from around the web. Along with STX’s own articles and updates, it’s a high-quality way to stay in the know.

Discover Your Perfect Python Match

The Python ecosystem is richer than ever in 2023. Take your skills to new heights by finding the Python newsletters that work for you. Whether you learn best from tutorials, news briefs, or deep dives, you’ll find your perfect match here. See you in the inbox!
